在文章中添加图片、音乐及视频

添加图片

  1. 首先更改_config.yml 中 post_asset_folder:true。当设置post_asset_folder为true参数后，在建立文件时，Hexo会自动建立一个与文章同名的文件夹，可以把与该文章相关的所有资源都放到这个文件夹，便于对每篇文章里的资源进行管理

  2. 在搭建的博客目录下打开Git bash ，执行:

    npm install hexo-asset-image –save

  3. 完成安装后用Hexo新建文章的时候会发现_posts目录下面会多出一个和文章名字一样的文件夹。图片就可以放在文件夹下面。

  4. 此时，就可以在文章任意位置添加如下代码格式即可插入图片：

    ![图片描述](新建文章名/图片名称.jpg)

  5. 在引入图片时，也可以在source文件夹中新建images文件夹，把图片统一放置在里面（然后删除新建文章时创建的文件夹），使用绝对路径插入图片：

    ![图片描述](/img/图片名称.jpg)

  6. 相对路径和绝对路径两种方法各有好处，根据个人习惯选择。
